# 创建第一个程序

在安装完成DevEco Studio之后，您可以通过创建第一个程序来验证安装是否成功。

1.  若首次打开**DevEco Studio**，请单击**Create Project**创建工程。如果已经打开了一个工程，请在菜单栏选择**File** > **New** > **Create Project**来创建一个新工程。
2.  选择**Application**应用开发，选择模板**Empty Ability**，单击**Next**进行下一步配置。更多模板的使用和说明请见[工程模板介绍](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-template)。
    
    ![](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251121142714.00292229832759785665402868450918:50001231000000:2800:9AECAC59097E735B2FC148D51A79BB0B0D39689EB2822C862D67870BBBD94F52.png "点击放大")
    
3.  进入配置工程界面，**Compatible SDK**表示兼容的最低API Version，此处以选择 **6.0.0(20)** 为例，其他参数保持默认设置即可。
    
    ![](https://alliance-communityfile-drcn.dbankcdn.com/FileServer/getFile/cmtyPub/011/111/111/0000000000011111111.20251121142714.89127214278646401899173208509862:50001231000000:2800:6A604F391561DE1B7EBD76938B9ED27D308E12B698C6A7CA289A668D0F796F60.png "点击放大")
    
4.  单击**Finish**，工具会自动生成示例代码和相关资源，等待工程创建完成。
